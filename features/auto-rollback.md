# Auto-rollback

{% hint style="info" %}
Available on the **Starter** and **Growth** plans.
{% endhint %}

Auto-rollback is what saves you the cleanup after a sale. When you set an end time on a launch, Launch Flow restores the original snapshot values at that moment — prices return to normal, the compare-at markdown clears, and (for theme launches) you can schedule the previous theme to republish.

### When rollback runs

* At the end date and time you set on the launch.
* It restores from the snapshot taken before the launch went live.

{% hint style="warning" %}
**Auto-rollback requires a paid plan.** On the Free plan, price changes are applied but won't revert automatically — you'd restore them manually. Upgrade to Starter or Growth to schedule an automatic end time.
{% endhint %}
