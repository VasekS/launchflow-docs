# Permissions & data

Launch Flow asks for the minimum access it needs to do its job. Here's what each permission is used for:

| Access                            | Why it's needed                                                          |
| --------------------------------- | ------------------------------------------------------------------------ |
| Edit products & collections       | To apply scheduled price changes and restore the originals on rollback.  |
| Edit Online Store pages & theme   | To publish scheduled page updates and themes at launch time (Growth).    |
| Store owner details               | To send account and notification emails to the right address.            |

For full details on how data is handled, see the [Launch Flow Privacy Policy](https://vaseks.github.io/launchflow-privacy).
